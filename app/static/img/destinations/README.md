# Как добавить реальные фото городов

## Куда класть файлы

```
app/static/img/destinations/
├── samarkand/    ← фото Самарканда сюда
├── bukhara/      ← фото Бухары сюда
├── khiva/        ← фото Хивы сюда
├── tashkent/     ← фото Ташкента сюда
├── fergana/      ← фото Ферганской долины сюда
├── shahrisabz/   ← фото Шахрисабза сюда
├── nukus/        ← фото Нукуса сюда
└── termez/       ← фото Термеза сюда
```

## Требования к файлам

- Форматы: `.jpg`, `.jpeg`, `.png`, `.webp`
- Имя файла любое (например `registan-1.jpg`, `photo1.jpg`, `IMG_0001.jpg`)
- Рекомендуемый размер: минимум 1200×900px, лучше 1920×1080px для главного фото
- Чем больше фото в папке — тем больше вариантов карточек на сайте (галерея,
  карточки достопримечательностей и т.д. автоматически используют разные файлы)

## Как это работает

Сайт **автоматически** определяет, есть ли реальные фото в папке города:

- Если фото есть → показывается реальное фото
- Если фото нет → показывается красивая SVG-иллюстрация (как сейчас)

**Никакого кода менять не нужно.** Просто закинь файлы в нужную папку,
закоммить и запушь — и фото появятся на сайте.

## Порядок действий

1. Собери фото по городам, разложи по папкам выше на своём компьютере
2. Скопируй папки в `app/static/img/destinations/` в проекте
3. В терминале:
   ```
   git add .
   git commit -m "Add real destination photos"
   git push
   ```
4. Готово — Railway задеплоит автоматически, фото появятся на сайте

## Сколько фото нужно на город

- Минимум: 1 фото (будет использоваться везде)
- Хорошо: 3-5 фото (для разных карточек — hero, галерея, достопримечательности)
- Отлично: 8-12 фото (полноценная фотогалерея на странице города)
